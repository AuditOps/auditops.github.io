# Privacy Policy for AuditOps Code Review  
**Effective Date:** March 23, 2026  

AuditOps LLC (“AuditOps,” “we,” “us,” or “our”) operates the AuditOps Code Review application (the “Application”), a GitHub Marketplace integration designed to verify pull request approvals and provide audit evidence for secondary review. This Privacy Policy explains how we collect, use, disclose, and protect information when you use the Application.

---

## 1. Information We Collect

### a. Information from GitHub
When you install or use the Application, we may access and process certain data from your GitHub account or organization, including:

- Repository metadata (e.g., repository names, IDs, and settings)  
- Pull request data (e.g., titles, descriptions, timestamps, approval status)  
- Commit metadata (e.g., commit IDs, authors, timestamps)  
- Reviewer activity (e.g., approvals, comments, review history)  

We only access the minimum data necessary to perform the Application’s functionality.

### b. Account and Contact Information
If you interact with us directly (e.g., support requests), we may collect:

- Name  
- Email address  
- Organization name  

### c. Usage Data
We may collect limited operational data such as:

- Log files (e.g., timestamps, error logs)  
- Application performance metrics  
- Configuration settings related to audits  

---

## 2. How We Use Information

We use collected information to:

- Verify that pull requests meet approval requirements  
- Generate and store audit evidence for compliance and secondary review  
- Provide insights and reporting related to code review processes  
- Maintain, improve, and secure the Application  
- Respond to customer inquiries and provide support  

We do **not** use repository data for advertising or marketing purposes.

---

## 3. Data Retention

We retain data only as long as necessary to:

- Provide audit evidence and reporting  
- Meet legal, compliance, or contractual obligations  
- Resolve disputes and enforce agreements  

Users may request deletion of their data at any time (see Section 7).

---

## 4. Data Sharing and Disclosure

We do **not** sell or rent your data.

We may share information only in the following cases:

- **With service providers** who help operate the Application (e.g., hosting, logging), under strict confidentiality obligations  
- **For legal reasons**, such as compliance with applicable laws, regulations, or legal processes  
- **To protect rights and security**, including preventing fraud or misuse  

---

## 5. Data Security

We implement reasonable administrative, technical, and organizational safeguards to protect data, including:

- Encryption in transit  
- Access controls and least-privilege principles  
- Monitoring and logging of system activity  

However, no system can be guaranteed 100% secure.

The Application integrates with GitHub and relies on its APIs. Your use of GitHub is subject to GitHub’s own privacy practices. We are not responsible for third-party services.
